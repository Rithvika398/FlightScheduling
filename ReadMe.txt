ALl the files required are located in 'src' folder (Ignore rest)


1. Adding flight 
To add flight compile the src/AddFlight.java
Commands:
cd inside src folder
javac AddFlight.java
java AddFlight



2.Scheduling Flight
To schedule flight i.e. to run the flight, compile src/Main.java
Commands:
cd inside src folder
javac Main.java
java Main




Note:
Whenever you are getting unwanted behaviour just delete all the files in src/File
and then copy all the files from 'File Copy' to File/
